# project-107cf1

This project is my research platform for Rust/Vulkan computer graphics and game development.

## Dependencies

- VulkanSDK 1.+

### Windows
- Visual Studio 2019
- LLVM

### Ubuntu
- sudo apt install build-essential
- sudo apt install xorg-dev
- sudo apt install llvm-dev libclang-dev clang

### macOS
- brew install llvm
